## 练习在其他分支上

> git 常用命令

```bash
简洁模式：git log --pretty=oneline

查看历史记录的版本号：git reflog

回档：git reset --hard 7c77ab0 ✨

回退版本1：git reset --hard HEAD^1

回退版本2：git reset --hard HEAD~2

还原文件：git checkout +文件名   【日志记录还在】

创建git：git init

查看文件状态：git status

查看日志：git log


查看分支：git branch -v

创建分支：git branch +分支名称

切换分支：git checkout +分支名

删除分支：git branch -d +分支名称

合并分支
  1. 切换到主分支
  git checkout master
  2. 合并
  git merge +分支名
  
查看冲突：git diff

解决冲突：切换到哪个目录，然后查看文件 vi +文件名
```
